Configuring your Intelligent Interface

It is necessary to configure your Intelligent Interface to make sure that it is using this server and a suitable printer ID.

To do this, please open the cloudPRNT configuration we bage of your Star Printer. The correct CloudPRNT URL depends on how you are running/accessing this server. If the server is running on your private PC then the URL should be: http://[serverIP]:8080/starcloudprnt and you can use any one of the prepared printer ID's visible on the printer list page. If you are acessing the SME hosted version of this server then the cloudPRNT URL is http://cloud.stareu.net/starcloudprnt

.

Firewall Checking

If you have trouble acessing this server from other computers, or using the Intelligent Interface, then please check the firewall settings of the PC that you are using to run the server. It is necessary to allow incoming tcp connections on port 8080.

Also please not that it is necessary to run the server as Administrator, for Windows to allow it to receive web requests.

Security

This server does not implement security features for the demo. However cloudPRNT does fully support http authentication, and https/TLS encryption.
